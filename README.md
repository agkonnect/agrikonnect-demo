# AgriKonnect — Interactive UI Demo

A complete 9-page interactive HTML prototype of the AgriKonnect agricultural marketplace platform.

## Pages

| Page | File |
|------|------|
| 🏠 Homepage | `index.html` |
| 🔐 Login / Register | `AgriKonnect_Auth.html` |
| 🛒 Browse Produce | `AgriKonnect_Browse_Page.html` |
| 🍅 Product Detail | `AgriKonnect_Product_Detail.html` |
| 💳 Cart + Checkout | `AgriKonnect_Cart_Checkout.html` |
| 🌾 Farmer Dashboard | `AgriKonnect_Farmer_Dashboard.html` |
| 📦 Buyer Dashboard | `AgriKonnect_Buyer_Dashboard.html` |
| 📊 Price Intelligence | `AgriKonnect_Price_Intelligence.html` |
| 📋 Post a Request | `AgriKonnect_Post_Request.html` |

## Navigation
Every page has a floating green menu button (bottom-right corner) to jump between all pages.

## Tech
- Pure HTML + Tailwind CSS (CDN)
- Vanilla JavaScript
- No build step required
